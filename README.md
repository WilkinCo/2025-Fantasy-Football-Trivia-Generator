# 2025-Fantasy-Football-Trivia-Generator

## Fantasy Football trivia game that generates NFL and fantasy football questions about the 2025 season from PFR's player data using Claude API

-------------------------------------------------------------------------

## Features

- Uses real NFL / fantasy stats from CSV datasets
- AI-generated trivia questions (Claude API)
- Difficulty levels (Easy / Medium / Hard)
- Position filtering (QB, RB, WR, TE)
- score and accuracy tracking

------------------------------------------------------------------------

## Stack
- Python
- Pandas (data processing)
- Streamlit (UI)
- Anthropic Claude API (question generation)
------------------------------------------------------------------------
## Setup
- Install Dependencies: pip install pandas streamlit anthropic
- Set API: $env:ANTHROPIC_API_KEY="your-api-key-here"
- Run: streamlit run app.py
